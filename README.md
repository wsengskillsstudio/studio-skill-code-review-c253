# Code Review Skill

This APM skill performs structured, multi-dimensional code reviews on diffs, pull requests, commits, or individual files. It systematically checks for logic errors, security vulnerabilities, performance issues, style inconsistencies, and test coverage gaps, then produces a severity-ranked report with actionable suggestions and positive highlights.

Use it whenever you need a thorough, constructive review of code changes — whether for a feature branch before merging, a security audit of sensitive code paths, or a sanity check on refactoring work. The skill adapts its analysis based on language, framework, and optional focus areas you specify, making it suitable for projects across the stack.
